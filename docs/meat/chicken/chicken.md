# 鸡肉

## 可乐鸡翅

|材料|分量|备注|
|---|---|---|
| 鸡翅 | 适量 | 洗净，正反面斜切2刀切口 |
| 生姜 | 适量 | 切丝 |
| 料酒 | 适量 |  |
| 老抽 | 适量 |  |
| 生抽 | 适量 |  |
| 盐 | 适量 |  |
| 可乐 | 1瓶 |  |
| 白芝麻 | 适量 |  |

* 腌制：`姜丝`、2`料酒`、1`老抽`、1`生抽`、少许`盐`；腌制20分钟。
* 热油，中火，煎鸡翅至两面金黄。
* 加2`料酒`、1`老抽`、1`生抽`；翻炒至均匀上色。
* 倒入`可乐`煮沸，转中小火焖煮10分钟；大火收汁，不停搅拌。【怕太甜可以放点盐中和甜味】
* 留少许汤汁后关火盛出；撒上`白芝麻`。

## 蒸蛋

|材料|分量|备注|
|---|---|---|
| 鸡蛋 | 2个 |  |
| 温水 | 适量 |  |
| 盐 | 适量 |  |
| 油 | 适量 |  |
| 葱花 | 适量 |  |
| 姜油 | 适量 |  |

* `鸡蛋`加`盐`、`油`、`温水`打散，过滤蛋液。
* 碗上裹`保鲜袋`，`牙签`扎孔。
* `热水`炖10分钟，加`葱花`炖2分钟，加`酱油`。
